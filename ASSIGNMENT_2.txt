Q.3 what is the purpose of an else block?

ANS: The `else` block provides an alternative action when the `if` condition is **false**.Without `else`, nothing happens when the condition is false.

--->Think of it as a fork in the road:

* If the condition is true → take one path.

* Otherwise (`else`) → take the other path.



Q.4 what is an else block executed?

ANS: In **C++**, the `else` block executes when the condition in the `if` statement is **false**.

---> Example:

#include <iostream>
using namespace std;

int main() {
    int number = 5;

    if (number > 10) {
        cout << "Number is greater than 10";
    }
 else {
        cout << "Number is 10 or less";
    }

    return 0;
}
`

# Output:

```
Number is 10 or less
```

--->Explanation:

* The condition `number > 10` is checked.

* Since `5 > 10` is **false**, the `else` block is executed.

* Therefore, `"Number is 10 or less"` is printed.

Q.21 What is a switch statement?

ANS: A **switch statement** is a control statement used to execute one block of code among many based on the value of a variable.

--->Example:

#include <iostream>
using namespace std;

int main()
{
int day = 2;


switch(day)
{
    case 1:
        cout << "Monday";
        break;
    case 2:
        cout << "Tuesday";
        break;
    default:
        cout << "Invalid day";
}

return 0;


}
